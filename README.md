# gwv

GlyphWiki data validator - http://ku6goma.appspot.com/gwv/


## Installation

```sh
python2 setup.py install
```


## Usage

```sh
gwv /path/to/dump_newest_only.txt
```

（↑を実行すると `dump_newest_only.txt` と同じディレクトリに `gwv_result.json` が生成される（フォーマットは今後大きく変更する可能性がある））
